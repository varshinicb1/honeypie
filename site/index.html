We'd rather tell you the truth than lose your trust

HoneyPie is an early, open-source build. Here's exactly what's real today and what isn't — no vague marketing language.

✅ Real today

  • Native Android / Jetpack Compose apps get real builds, real installs, and real screen captures — verified end-to-end against Google's own Sunflower sample app
  • Drag-and-drop a folder or paste any git repo URL
  • Zero setup — Node.js is bundled inside the installer
  • Fully open source, MIT-style, on GitHub

🚧 Not there yet

  • Flutter and iOS projects currently get placeholder assets, not real captures
  • Installer is unsigned — expect a SmartScreen warning
  • Needs the Android SDK + an emulator already set up for real capture
  • No AI-driven exploration or copywriting yet — today's logic is deterministic, not AI
