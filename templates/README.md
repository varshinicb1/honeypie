# Templates (Index)

This folder is a placeholder for exported/standalone copies of theme `template.yaml` scene-graph definitions once implementation begins (see `../docs/13-template-system.md` for the full specification and inline examples).

At the specification stage, template examples live inline in `../docs/13-template-system.md`. Once `plugins/theme-*` packages exist in the implementation repository, this folder's role shifts to holding curated example templates referenced by the docs site and by `honeypie plugins add` scaffolding.
